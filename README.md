# Kirby Simplify

**Kirby Simplify** automatically generates simplified language versions of your content, making your website more accessible and inclusive. Perfect for people with reading difficulties, non-native speakers, and anyone seeking easier access to your content.

## Features

- **AI-Powered Translation** - Supports OpenAI, Anthropic, Google Gemini and Mistral AI
- **Community-Approved Models** - Curated list of tested AI models for optimal translation quality
- **Pre-configured Rules for 32 Languages** - Following European (FALC, DIN SPEC 33429, UNE 153101) and international (ISO 24495-1, DIN 8581-1) standards
- **Multiple Language Variants** - Support for multiple simplification levels per language
- **Flexible Provider Setup** - Use different AI providers and models for each language variant
- **Smart Translation Modes** - Automatic or manual translation per page
- **Highly Configurable** - Customize prompts, field types, and privacy settings
- **Budget Control** - Set daily/monthly limits and track costs per provider
- **Real-time Monitoring** - View translation jobs, reports, and statistics
- **Beautiful Panel UI** - Fully integrated into Kirby's admin interface with dark mode support, available in 32 languages
